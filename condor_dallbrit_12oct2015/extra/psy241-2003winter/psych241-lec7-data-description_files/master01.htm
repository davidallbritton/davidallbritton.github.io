Prepare data sheets – if the subject has filled in information (such as ratings in a survey) transfer that information to the margin where it will be easy to read when entering data into a summary data sheet
One data point per row – book says one subject per row, but with repeated measures (within subs design) each sub could have more than one data point.  Never have 2 subjects on a row, though.
Coding guide – specifies the format for the data file on a computer (number of columns for each variable and order of variables) and documents the meaning of the codes entered into the data file (1=male, 2=female for example).
**Best to start with step 3 – create a coding guide – when you are designing the experiment.  Steps 1 and 2 can be skipped if data is collected by computer *and* you have used the coding guide when creating the data collection program so that the data will already be in the form specified by the coding guide.
Step 5 – missing data, outliers etc – involves preliminary (exploratory or descriptive) data analysis before doing your inferential stats.  You may have to edit the data file to eliminate invalid data, mark missing data, and eliminate outliers.
