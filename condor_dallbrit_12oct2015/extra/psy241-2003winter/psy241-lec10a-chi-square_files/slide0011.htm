Performing the Chi-Square Test
1.Name the variables using the variables tab in the SPSS data window
2.analyze -> descriptive statistics -> crosstabs
3.Use arrow button to move “gender” into “rows” box
4.Use arrow button to move “footware” into “columns” box
5.Click “Statistics” box
6.Check the box for “Chi-square”, then click “Continue”
7.Click the “Cells” box.
8.Under “Percentages” check the boxes for “Row” and “Column”
9.Click “OK”
