Symbolic Distance Effect

 

Instructions for lab report

 

For this lab report, you will analyze the group data from the lab exercise, plot the group data means (provided by the instructor) on the graphs in the MEL Lab workbook, and write a manuscript describing the study in APA format.  You should include the graphs from the workbook as tables or figures in your paper.

 

On the course web page you will find three data files to use for the analyses, along with some tables of means that you may find helpful for completing the workbook graphs.  In the tables, "conditio" is the question (0="which is larger"; 1="which is smaller") and "largersm" is the pair type (1=both large; 2=both small; 3=one of each).  The data in the Table is for correct responses only, as are the data in the data files.

 

The analyses you are to do are the following:

  • Repeated measures ANOVA for the effects of question type (“Which is larger” [q0] vs. “Which is smaller” [q1]) and type of animal pair (p1=both large, p2=both small, p3=one of each). 
    • Data file: lab2_5-symbdist-question-condition.dat
    • The data are arranged in this fashion:
      • One line per subject, subject number is the first variable
      • Then with 6 mean RTs (one for each condition) for each subject
      • Pair type is nested in question type: 
        q0p1 q0p2 q0p3 q1p1 q1p2 q1p3
    • Download the data to your computer (right click and save to disk)
    • Import the data to SPSS (file type = “data”, variable names in first row)
    • Run a repeated measures ANOVA (analyze, general linear model, repeated measures)
    • Report the main effects (question and pair) and the interaction (q*p)
  • Repeated measures ANOVA for the effects of ordinal distance between animals in each pair (distance = 1 to 11)
    • Data file: lab2_5-symbdist-distance-for-anova.dat
    • The data are arranged in this way:
      • One line per subject, subject number is the first variable
      • 11 mean RTs per subject, for the 11 ordinal distances between animals in a pair.
    • Download the data file and import to SPSS
    • Run a repeated measures ANOVA
    • Report the main effect of distance, and the test of linear trend
  • Regression of distance on RT
    • Data file:  lab2_5-symbdist-distance-for-regression.dat
    • The data are arranged in this way:
      • 3 columns with variable names in first row:
        subject, distance, resprt
        (subject, distance, reaction time)
      • Do a linear regression of distance on resprt:  analyze, regression, linear
      • Report the correlation between distance and RT, and report how much the responses speed up with each increase in ordinal distance (in other words, the slope of the regression equation, which is reported as the unstandardized Beta weight, or “B”).
