Duration of the Icon

Analysis Instructions for Instructor to prepare data files

 

Instructor’s notes:

 

1)  Provide the following tables for students to plot on the graphs in the workbook:

 

  • Table 10:  Percent recalled in exact order, averaged across subjects
  • Table 11:  Percent recalled in any order, averaged across subjects
  • Table 20:  Percent recalled in exact order, for each subject
  • Table 21:  Percent recalled in any order, for each subject

 

After running all these tables in mellab (“sanalyze”), rename the file “icon.out” to “lab1_.2-tables.txt” and give to students (place in the “labs” directory), after editing it to clean it up and make clear to students which condition is which (1=partial report, 0ms delay;  2=partial report, 150ms delay; 3=partial report, 500ms delay; 4=full report, 0ms delay)

 

2) Create data file for students to use.

  • Output data specifications form 20:  Data by conditions. 
  • First do it using “Form ID Number 1:30” for proportion recalled in exact order: Function-3 to print to file.
  • Then do it using “Form ID Number 1:31” for proportion recalled in any order: Function-3 to append to file.
  • The output file is “icon.out”.  Rename it to “icon-data.txt”.
  • Reboot to linux, then run “get-lab1_2-files” to create data files for students to analyze.
  • The resulting data file is lab1_2-data.txt.   Note that the data in the table is the mean of each subject (averaging across multiple attempts at the experiment). 

 

3)  Create SPSS data files for students to analyze, lab1_2.sav, and post to web page (or have them import the data themselves to learn how to do it).

 

 

4)  Place the files lab1_2-data.txt, lab1_2-tables.txt, and lab1_2.sav, if it exists, in the course web “labs” folder.
