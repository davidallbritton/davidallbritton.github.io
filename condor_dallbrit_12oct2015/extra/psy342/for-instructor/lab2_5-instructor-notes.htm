Lab 2.5

Symbolic Distance Effect

Analysis Instructions

 

Instructor’s notes:

 

1)  Provide the following tables for students to plot on the graphs in the workbook:

 

  • Table 10:  RT by question [“conditio”] (0=which is larger; 1=which is smaller) and pair type [“largersm”] (1=both large, 2=both small, 3=one each).  Correct responses only.

 

  • Table 30:  RT by ordinal distance (1-11) between animal pairs

 

After running both these tables in mellab (“sanalyze”), rename the file “symbdist.out” to “lab2_5-tables.txt” and place it in the "labs" directory on the course web site.

 

2) [optional]  The following plots may be useful for grading the plots students make as lab reports.  The graphs they are required to do are Plot 10 and Plot 70.  Plots 11, 12, and 50 can be provided to students before they do the lab report.

  • Plot 10:  interaction of question (“larger?” vs “smaller?”) and pair type (both large, both small, one of each)
  • Plot 70:  RT as a function of ordinal distance between animal pairs
  • Plot 11:  main effect of pair type
  • Plot 12:  main effect of question type
  • Plot 50:  frequency polygon of all response times combined

 

3) [optional]  Run the two anovas in MEL to check the spss results against later

  • Anova form 10 – condition by question type, DV = RT correct
    • Run in sanalyze
    • Run “anova”
    • Rename “symbdist.out” to “symbdist-anova1.txt”
  • Anova form 20 – RT correct as a function of ordinal distance between stimulus pairs
    • Run in sanalyze
    • Run “anova”
    • Rename “symbdist.out” to “symbdist-anova2.txt”

 

3) Create data file for students to use.

  • Output data specifications form 10:  all variables and all trials data.  Function-3 to print it out to “symbdist.out”.  Then rename to to “symbdist-data.txt”.
  • Reboot to linux, then run “get-lab2_5-files” to create data files for students to analyze
  • Copy the 3 resulting ".dat" files to the labs folder on the web.
