Balanced Latin Squares

Create an applet for generating balanced latin squares of a given size. Tasks:

randomly order the N conditions;

generate the square: 1, 2, N, 3, N-1, 4, N-2, etc for first row, then add one to rotate through conditions for each subsequent row.

If N is odd, reverse each row to create a second square to complete the counterbalancing

print out the squares

Could also block randomize the order of the rows for however many subjects are planned - some multiple of N (or of N*2 for an odd number of conditions).

How many conditions are there in your design?
