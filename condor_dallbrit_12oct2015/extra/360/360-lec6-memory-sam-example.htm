A simplified Example of recognition in SAM: 

Recognition judgments for Item 1 (dog) and for an unstudied item (pig)

Study list = dog, rock, house, cat, horse

Recognition test = dog, pig, ....  ("dog" = item 1, "pig" = unstudied item)


Overview of the process:

  • Encoding:  strengths are built up for:
    1. Item 1's self-strength:  the self-strength parameter c * the length of time, t(1) that item 1 is in the STM buffer.
    2. Item 1's context-strength:  the context parameter a * the length of time t(1) that item 1 is in the buffer.
    3. Item 1's inter-item strength with other items in the buffer.  Item 1's inter-item strength with item 2, for example, is the inter-item strength parameter b * the length of time that item 1 and item 2 were in the buffer together, t(1,2).  For any item that is never in the buffer at the same time as item 1, such as item 5, the inter-item strength is zero (in the model, a residual strength value of r is used instead of zero).
  • Recognition -- for each test item:
    1. Form a set of cues in STM (a memory probe) consisting of:
      • the test item
      • the context
      • (these are left out in order to simplify this example) the item category, other test items that still in STM during the recognition test.
    2. Match the probe to all items in memory to determine its familiarity
      • Familiarity = (Sum of strengths for context) * (Sum of strengths for the item)
    3. Compare the familiarity to a decision criterion (a cutoff).
      •  If familiarity is above the criterion, say "yes, I recognize this item."  If familiarity is below the cutoff, say "no, I do not recognize this item."  
      • (Does this remind you of anything?  Think about sensory thresholds).

Doing the Calculations:  

Assumptions:

  • Study time = 2 seconds per item.
  • Assume all 3 parameters (a, b, c) are set to 1 for this example.
  • r = 1
    r = residual strength between items that were not in STM together.  Typically this is set to some low value, but we will use 1 here for ease of calculation.
  • Criterion = 500
    This is the familiarity criterion for the recognition decision.  Any test item with a familiarity above this value gets a "yes" respone; below this value it gets a "no" response.

Test item 1 ("dog"):

  • Probe = "dog" & context
  • Familiarity ("dog") = (Sum of strengths for "dog") * (Sum of strengths for context) = 800
    • Amount of time in STM buffer:
      • t(1) = 8 seconds -- assuming 4 items at a time can be in the STM buffer, item 1 would remain in the buffer for a total of 8 seconds before it was displaced.
      • t(2,1) = 6 seconds -- the time that both "dog" and "rock" (items 1 and 2) are in the buffer together
      • t(3,1) = 4 seconds -- "dog" and "house" (items 1 and 3), and so on..
    • Sum of strengths for "dog" = c*t(1) + b*t(2,1) + b*t(3,1) + b*t(4,1) + b*t(5,1) = 8 + 6 + 4 + 2 + 0 = 20
    • Sum of strengths for context = a*t(1) + a*t(2) + a*t(3) + a*t(4) + a*t(5) = 8 + 8 + 8 + 8 + 8 = 40
    • Familiarity ("dog") = 20 * 40 = 800
  • The familiarity of "dog" is greater than the criterion (800 > 500), so answer "yes"

Unstudied test item ("pig"):

  • Probe = "pig" & context
  • Familiarity ("pig") = (Sum of strengths for "pig") * (Sum of strengths for context) = 200
    • Sum of strengths for "pig" = 5*r = 5
    • Sum of strengths for context = a*t(1) + a*t(2) + a*t(3) + a*t(4) + a*t(5) = 8 + 8 + 8 + 8 + 8 = 40
    • Familiarity ("pig") = 5 + 40 = 200
  • The familiarity of "pig" is less than the criterion (200 < 500), so answer "no"
