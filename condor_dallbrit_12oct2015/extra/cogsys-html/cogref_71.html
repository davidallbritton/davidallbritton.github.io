2.3.7.2 Wait for J1B1 (+/- J2B1) (#L)

The Wait for J1B1 (+/- J2B1) command pauses Cogsys execution until gameport button J1B1 is pressed alone or is pressed with J2B1.

Name:
Wait for J1B1 (+/- J2B1)
Class:
Game Port
Code:
#L
Syntax:
#L
System:
The Wait for J1B1 (+/- J2B1) command pauses Cogsys execution until gameport button J1B1 is pressed alone or is pressed with J2B1.

Examples:

input
#L
subject

cogsys
Keeps waiting.
subject
Presses J2B1 as well as J1B1
cogsys
Continues.
