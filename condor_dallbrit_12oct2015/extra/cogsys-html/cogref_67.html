2.3.6.3 Scope Response (#Y)

The Scope Response command displays a string of characters on the scope and waits for a response. The text is written to the scope with the attributes set by Set Scope Attributes. It writes the key and time to output.

Scope Response will terminate Cogsys execution of the experiment if the (BREAKOUT_CHAR) key is entered.

Name:
Scope Response
Class:
Scope
Code:
#Y
Syntax:
#Y[text]
text can not contain the `]' character
System:
Displays
Output:
The key pressed and time are written to output.
Breakout:
YES

Test Files:
`SCOR_T01.IN'
tests basic operation

Examples:

input
#Y[hi there!]
cogsys
Writes the string `hi there!' on the scope with the current scope text attributes.
subject
Press the z key after 100 milliseconds.
output
z100
