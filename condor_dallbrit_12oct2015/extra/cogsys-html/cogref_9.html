1.2.5 Installation Problems

If you did not see the welcome message, something is wrong with your installation. Some things to consider:

  • Not enough conventional memory - make sure that no TSR (terminate and stay resident) programs are loaded; look particularly for mouse and other device drivers.
  • Faulty installation. For best results, always install Cogsys to a "clean" directory, and always install from the `COGINST.EXE' archive. Simply copying the `COGSYS.EXE' executable file from another installation may not work. It is possible that files from a previous installation are interfering with the latest program.
  • Restart the computer.

If none of these suggestions work, make a note of the problem and send an email describing what's wrong to usman@thynne.psych.nwu.edu.
