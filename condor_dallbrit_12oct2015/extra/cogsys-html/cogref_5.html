A note on Windows

Cogsys is real time software; simply put, timing matters. Because most modern operating systems (Unix, Windows 95/98/NT, MacOS) are multitasking (that is, they can and do run more than one program at a time) Cogsys can not run reliably in these environments, because the operating system may decide at any moment to run a different process.

However, Cogsys will run correctly on computers with Windows 95/98 if they are in MS-DOS mode (not a "DOS box"). To properly run Cogsys, a Windows system must be shutdown with the Restart in MS-DOS mode option first.
