2.3.4.2 Text-Mode Escape Character (\x)

Escape Character command allows display of characters that would otherwise be interpreted as part of a Cogsys command. Cogsys reads the character following the escape character and displays it.

This command only works in a text mode. It is not needed (and will not work) within any other command, including Graphics-Mode Display Text (2.3.4.3 Graphics-Mode Display Text (#G)).

Name:
Text-Mode Escape Character
Class:
Display Text
Code:
\
Syntax:
\char
System:
Displays the character char at current cursor location in text mode.

Requires:
Cogsys must be in a text mode.

Test Files:
`ESCP_T01.IN'
tests basic operation
`ESCP_B01.IN'
benchmarks the display of 1000 escaped characters

Examples:

input
Use the \#R command to get a response
cogsys
Writes `Use the #R command to get a response' on the screen.
