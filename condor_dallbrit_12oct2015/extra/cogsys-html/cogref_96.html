3.3.4 RANDEXT

The RANDEXT extension can create several types of pictures, even series of pictures, which involve not only random dots but also checkerboards and solid colors. Complex extensions like RANDEXT split their functionality into experiments (named so because each functionality was usually implemented for specific experiment) and the first argument to the extension is an identifying experiment number.

This is the syntax of the various forms of the RANDEXT:

Syntax: RANDEXT exptnum [ arg1 ... ]
