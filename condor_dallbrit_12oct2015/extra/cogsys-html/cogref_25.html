2.3.1.1 Wait (#W)

The Wait command takes one argument that specifies the number of milliseconds to wait.

Name:
Wait
Class:
Control
Code:
#W
Syntax:
#Wwaittime
waittime is a timelimit
Test Files:
`WAIT_T01.IN'
tests basic operation
`WAIT_B01.IN'
times the #R-#W-Display loop overhead
Examples:
input
#W100
cogsys
waits 100 milliseconds
input
#WV1
cogsys
waits for the number milliseconds specified in the variable V1
