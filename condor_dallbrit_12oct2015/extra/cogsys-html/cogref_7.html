1.2.3 Installation

Cogsys is very easy to install: simply run the self-extracting ZIP archive anywhere on your hard drive. It does not make any modifications to any system files. The files will unpack in the current directory. The rest of this document will assume that you have unpacked the Cogsys hierarchy in a directory called `C:\COGSYS' on your hard drive. If you put Cogsys somewhere else, substitute that location for `C:\COGSYS' throughout these instructions.

Because Cogsys version 3.0.x runs in real mode, it requires as much conventional memory as possible. The line DEVICE=HIMEM.SYS should be the only line in your `CONFIG.SYS' start up file. You may find it easiest to create a new MS-DOS 6.2 configuration profile for Cogsys so that you can easily switch between Cogsys and other settings you have. Consult any reference on MS-DOS for more information on how to do this.
