1.2.2 Getting Cogsys

Cogsys is available in self-extracting ZIP format from:

 
http://wally.psych.nwu.edu/computing/software/inhouse/cogsys/cogsys.html

That site will always have the latest stable version, release notes, and documentation. Follow the links to the COGINST.EXE file. You can also use any ftp client to directly get the file from ftp://wally.psych.nwu.edu/pub/archive/coginst.exe.

Alternatively, you may have a 3.5" floppy disk called Cogsys 3.0 Install Disk. In that case, simply copy the `COGINST.EXE' archive file from your `A:' drive.
