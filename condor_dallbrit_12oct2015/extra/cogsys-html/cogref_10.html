1.2.6 Uninstalling Cogsys

Because Cogsys makes no changes to system files, to undo the the work of the install program, simply delete the `C:\COGSYS' directory. In DOS, you can use the deltree command.

Caution: The deltree command will permanently delete the contents of a directory.
