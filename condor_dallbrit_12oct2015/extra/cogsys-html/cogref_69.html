2.3.7 Game Port Commands

The gameport commands read input from the PC gameport. The notation JjBb is used to signify button b on joystick j. The buttonbox is a specialized input device which plugs into the game port. The buttonbox has four buttons which correspond to J1B1, J1B2, J2B1, JB22. Cogsys writes the values `A' through `D' for these inputs, respectively.
