3.3 Extension Reference

This setion first explains certain features that are common to all extensions. It then details the four extensions that come with Cogsys:

    SQUAREXT for drawing squares CIRCEXT for drawing circles RANDEXT for random dot experiments TXTEXT for letter experiments

For each extension, the full argument syntax is explained. In addition, simples notes on its implementation are included.
