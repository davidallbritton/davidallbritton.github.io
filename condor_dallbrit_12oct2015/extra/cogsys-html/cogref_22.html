2.2.2 Saving Ouput on Error

When Cogsys encounters an error, it prints the error information followed by this message:

 
STOP! This experiment has an error in it. Please wait until
the test supervisor returns. Do NOT respond to the question below.

Attention test supervisor: Save output file? (Y/N)

Entering `y' here here will save whatever output was collected to the output file.
