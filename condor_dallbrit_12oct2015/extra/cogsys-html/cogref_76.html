3. Extensions

Extensions provide a mechanism for the advanced user to extend the capabilities of Cogsys without having to modify or even have access to the main program's source code. In Cogsys 3.0.15, extensions are only used to dynamically create pictures for display, but this will almost certainly change in the future.

This chapter completely covers all aspects of using and creating extensions.

Introducing Extensions, provides an overview of extensions: what they are, what they can do, and how they work.

Extension Reference, is a user's reference guide to the extensions that ship with Cogsys. All user options are described along with a (simplified) explanation of how they were implemented.

Creating Extensions, describes how to design and implement new extensions (and how to modifiy and rebuild existing ones). This section is only of interest to programmers who are developing new extensions for their testlists.
