2.3.7.1 Wait Any Button (#K)

The Wait Any Button command pauses Cogsys execution until any gameport button is pressed.

Name:
Wait Any Button
Class:
Game Port
Code:
#K
Syntax:
#K
System:
Pauses Cogsys execution until any gameport button is pressed.

Test Files:
`WGMP_T01.IN'
tests basic operation

Examples:

input
#K
cogsys
Waits until the subject presses a gameport button.
