1.2.4 Testing the Installation

You're now ready to the test the installation. Cogsys comes with a large number of demonstration and diagnostic testlists in the `\cogsys\sample' directory. By convention, testlists have the `.IN' extension. Right now, you just want to run the `sample\hello.in' which is the simplest testlist possible.

Make sure you are at a MS-DOS prompt, and change to the `C:\COGSYS' directory. Then simply run Cogsys with the name of the teslist you want to run:
 
C:\> cd \cogsys
C:\COGSYS> cogsys sample\hello.in

You should briefly see a Cogsys version message, and then the screen will clear and you'll see the words:
 
Welcome to Cogsys!
Press any key to exit.
at the top of the screen. Press any key, and you'll see the standard Cogsys end of testlist message
 
End of experiment -------------------- Thank you.
followed by your `C:\' prompt.
