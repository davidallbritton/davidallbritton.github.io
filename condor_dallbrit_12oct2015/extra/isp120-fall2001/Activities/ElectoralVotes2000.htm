Activity

The purpose of this activity is to analyze election data.  In particular, we will look at the election of 2000, the most recent example where one candidate (George W. Bush) won the electoral vote, and another candidate (Al Gore) won the popular vote.  This occurred only 2 other times in history - the 1888 election of Benjamin Harrison and the 1876 election of Rutherford B. Hayes.  In the process, we will learn about the IF command in Excel, something we haven't yet looked at in Quantitative Reasoning.

1.  Open the file http://qrc.depaul.edu/Excel_Files/Election2000.xls.  It contains popular vote by state in the 2000 election, whose main presidential candidates were George W. Bush, the Republican candidate, and Al Gore, the Democratic candidat.  It also contains the number of electoral votes for each state at that time.  The number of electoral votes a state has is equal to the sum of the number of its representatives and the number of its senators, that is, 2 more than the number of representatives.  The number of representatives is roughly proportional to the population of the state.  There are 537 total electoral votes, so 270 are needed to win the election.  If a presidential candidate wins the popular vote in a given state, he receives all the electoral votes of that state.  This method raises the possibility that one candidate can have more votes than the other candidate, yet lose the election.  This scenario occurred in 2000.  We will explore how it came about.

Include in your Word document your answer to each question.

a.  What was the absolute number of votes received by Al Gore overall?  What was the absolute number of votes received by George W. Bush overall?   In column E, calculate the total number of popular votes cast in that state. (Hint:   In cell E4, type =C4+D4 ) In column F, calculate the percentage of the popular vote received by Al Gore in each state.  [NOTE:  We are neglecting any popular vote received by third party candidates.] What percentage of the popular vote did Gore receive OVERALL?  In column G, calculate the percentage of the popular vote received by George W. Bush.  What percentange of the popular vote did Bush receive OVERALL?   Who won the presidency ACCORDING TO THE POPULAR VOTE?

b.  Now we wish to calculate the electoral votes each candidate received.  The rule is simple:  if Gore's vote in a state is higher than Bush's vote, then Gore gets all the electoral votes of that state.  In order to do this easily in Excel (i.e. without doing each state manually), one needs a conditional statement, an  IF  statement.  The IF statement in Excel has the following syntax:

    =IF(logical condition, what happens when condition is true, what happens when condition is false)

For example =IF(B2>0,C2*2,D2+1) will produce C2 multiplied by 2 if B2 is greater than zero, and otherwise, if B2 is less than or equal to zero, it will produce D2+1.

Label column H, "Gore wins electoral votes."  In column H, write an IF statement such that if Gore has more popular votes than Bush, the number of electoral votes from that state will appear in column H, if Gore doesn't have more popular votes than Bush, a zero will appear in the column. 

Label column I, "Bush wins electoral votes."  In column I, write an IF statement such that if Bush has more popular votes than Gore, the number of electoral votes from that state will appear in column I.  If Bush doesn't have more popular votes than Gore, a zero will appear.

How many electoral votes does Gore win in total?  How many electoral votes does Bush win in total?  Who won the presidency ACCORDING TO THE ELECTORAL COUNT?  Compare this answer to your answer in part (a).

d.  We would like to understand how this unusual situation (winning the popular vote but losing the election) came about.  The key is the margin of victory in the states, especially the large states.   In column K, subtract the value in column F from the value in column G, to produce a column which we can entitle "Bush's margin of victory".  Note it is negative if Gore won.  Finish off by making a scatter plot (NOT CONNECTED) with electoral votes on the x-axis and margin of victory on the y-axis.  Add this graph to your word document.  (Be sure to label the axes and give it a good title.)
