Note that this slide is a simplification of the general topic of controlling for order and sequence effects in within-subjects designs, which will be covered in subsequent lecture material.
Also note that this is discussing counterbalancing for order and sequence effects, but you can counterbalance other things too:
•2 different experimenters
•2 genders (if you think there might be gender differences)
•2 different stimuli for the same condition
Order effects, example: if you get better at the task as the experiment goes along (regardless of condition), that would produce an order effect.   Example: if the task is to identify the color of a stimulus, you might get faster with practice (because you learn which keys to press, etc).  So Condition B would be faster if it was the second condition than if it was the first condition presented.
Sequence effects:  color perception example – if the IV is the color a test stimulus is presented in (red, green, or blue) and the DV is speed to identify its color, there would be sequence effects on “green” depending on which other color it followed.  Following “red” it will be easier to identify than following “blue”, because habituation to the color red biases receptors towards perceiving green.  So the sequence “red-then-green” would have an effect on responses to the condition “green”

Note that you can not completely* control for sequence effects with just 2 or a few repetitions of each condition.  To completely control for sequence effects with 2 conditions:
Ab ba
With 3:
Abc acb bca bac cab cba
With 4, you need 12*4 = 48 trials per subject…
Note that to completely* control for sequence effects within groups, you need n! subjects, where n is number of conditions.  To completely control for sequence effects with 2 conditions:
Ab ba
With 3:
Abc acb bca bac cab cba
With 4, you need 12*4 = 48 subjects per counterbalancing group…
So, full counterbalancing quickly becomes impractical.
