Counterbalancing in
Within-Subjects Designs
• Counterbalancing within subjects
– Useful when each level of the IV occurs multiple times
for each subject
– Each subject gets both sequences of the conditions
• Counterbalancing within groups (not within
each subject)
– Useful when there are many (more than 2) conditions
– Necessary if each condition occurs only once per
subject
– An equal number of subjects get each sequence of the
conditions
