SAM -- a mathematical model of episodic memory

    • Raaijmakers & Shiffrin, 1981; Gillund & Shiffrin 1984
    • Features:
      • STM = a limited capacity buffer
        • accesses knowledge from LTM
        • affects what is stored in LTM
      • LTM = a large set of interconnected concepts
      • retrieval:  cue-dependent
      • forgetting:  due entirely to retrieval failure -- there is no decay in LTM
      • context  is a crucial determinant of memory performance
    • The model:
      • LTM consists of images -- feature sets for things and events that include information about
        • the context of encoding of the item
        • item-specific information, such as the item's name, meaning, etc.
        • inter-item information, such as the fact that two items occurred at the same time in STM
          -- the connections between the item and context, and between the item and other items depends on how long the item was in STM.  An item that is in STM a long time will form stronger connections to the encoding context.  
      • retrieval
        • Retrieval is in response to a probe.  For recall, the probe consists of only the encoding context (eg: probe = "what words did you see in the list you just read?").  For recognition, the probe consists of the encoding context and the test item (eg: probe = "was the word 'dog' in the list you just read?").
        • The probe activates images as a function of
          1. the similarity of an image to the probe
          2. the strength of the connections between the image and the retrieval context, and the strength of the connections between the image and other images in memory.
