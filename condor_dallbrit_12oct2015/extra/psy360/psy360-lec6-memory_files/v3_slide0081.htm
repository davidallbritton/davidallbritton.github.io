Implicit Memory:
A Separate Memory System?
• Spared implicit memory in amnesia
• “Double-dissociation” of explicit (episodic)
memory and implicit memory (priming)
– Manipulations that affect explicit memory (e.g., depth
of processing) do not affect implicit memory
– Manipulations that affect implicit memory (e.g.,
physical similarity) do not affect explicit memory
– Some tasks (e.g., generation effect, Jacoby 1983) have
opposite effects on the two types of memory tests
