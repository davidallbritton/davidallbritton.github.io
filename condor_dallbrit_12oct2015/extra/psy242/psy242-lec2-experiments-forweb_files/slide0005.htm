Some Terminology
•IV = what the experimenter manipulates (varies) in an experiment; the hypothesized cause
•DV = what the experimenter measures to test the hypothesis in an experiment; the hypothesized effect
•Factor = IV
•Level = condition = treatment:
–One value of an IV
•Control Variable – value held constant
•Counterbalancing Variable
•Confounded Variable – covaries with IV
•Random variable – value is randomly varied
