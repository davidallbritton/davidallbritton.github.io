The purpose of this activity is to analyze more election data.  In particular, we will look at the election of 2000, the most recent example where one candidate (George W. Bush) won the electoral vote, and another candidate (Al Gore) won the popular vote.  This occurred only 2 other times in history - the 1888 election of Benjamin Harrison and the 1876 election of Rutherford B. Hayes.  In the process, we will learn about the IF command in Excel, something we haven't yet looked at in Quantitative Reasoning.

1.  Open the file 2000ElectionCertifiedResults.xls.  It contains popular vote by state in the 2000 election, whose main presidential candidates were George W. Bush, the Republican candidate, and Al Gore, the Democratic candidate.  It also contains the number of electoral votes for each state at that time.  The number of electoral votes a state has is equal to the sum of the number of its representatives and the number of its senators, that is, 2 more than the number of representatives.  The number of representatives is roughly proportional to the population of the state.  There are 537 total electoral votes, so 270 are needed to win the election.  If a presidential candidate wins the popular vote in a given state, he receives all the electoral votes of that state.  This method raises the possibility that one candidate can have more votes than the other candidate, yet lose the election.  This scenario occurred in 2000.  We will explore how it came about.

b.  Now we wish to calculate the electoral votes each candidate received.  The rule is simple:  if Gore's vote in a state is higher than Bush's vote, then Gore gets all the electoral votes of that state.  In order to do this easily in Excel (i.e. without doing each state manually), one needs a conditional statement, an  IF  statement.  The IF statement in Excel has the following syntax:

    =IF(logical condition, what happens when condition is true, what happens when condition is false)

For example =IF(B2>0,C2*2,D2+1) will produce C2 multiplied by 2 if B2 is greater than zero, and otherwise, if B2 is less than or equal to zero, it will produce D2+1.
