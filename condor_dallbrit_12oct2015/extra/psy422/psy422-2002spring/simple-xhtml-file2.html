How to edit a page on your local PC

You can create or edit a web page on your local PC and then upload it to your server using ftp. Follow these steps to edit this page and place it in your public_html folder on the server.

  1. In your browser menu, click on "File - Save as" and save this page to your local computer. Be sure you select "Web Page, HTML Only" under "Save as type" in IE, and not "Web Page, complete," or it will make the source file an unreadable mess. Use mypage1.html as the file name. (Put it on the desktop or in some other location on the hard drive where you can easily find it later.)
  2. Open the file you just saved with Notepad (usually found in the Windows Start Menu under "Programs - Accessories"). Be sure you use Notepad or some other plain text editor, and not Word or any other word processing software.
  3. Follow the instructions in the comments in the file.
  4. Follow the rest of the instructions on the Week 2 web page to upload the file.
